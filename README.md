## About

Xenzia is a web-based snake game.

I built it to solidify my understanding of JS fundamentals and because playing the game is nostalgic, for me.

### Hint

Eat the food but don't hit the walls, your tail or your body.

## Stack

* HTML
* CSS
* Vanilla JS
